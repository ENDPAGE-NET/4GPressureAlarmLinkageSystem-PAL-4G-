# PAL_4G 部署文档

目标环境：**Linux + 宝塔面板**，前后端分域部署。

## 1. 部署架构

```
usr-iot.endpage.net（前端）         usr-iot-backend.endpage.net（后端）
    │ 宝塔 HTML 纯静态站点              │ 宝塔 Python 项目
    │ web/dist/                         │ backend/
    │                                   │ uvicorn :8001
    │                                   │
    └── 前端 JS 直接请求 ──────────────> │ /api/v1/*
                                        │
                                        │ EMQX Broker (同机)
                                        │ :1883 (MQTT)
                                        │ :18083 (管理面板)
```

## 2. 域名与端口

| 服务 | 域名 | 端口 | 用途 |
|------|------|------|------|
| Web 前端 | `usr-iot.endpage.net` | 443 | 纯静态站点 |
| 后端 API | `usr-iot-backend.endpage.net` | 443 → 8001 | FastAPI，宝塔反代 |
| MQTT Broker | `usr-iot-backend.endpage.net` | 1883/8883 | 设备接入（与后端共用域名） |
| EMQX Dashboard | 服务器内网 | 18083 | 管理面板，不公开 |

## 3. 部署步骤

### 3.1 后端

宝塔面板 → Python 项目，域名 `usr-iot-backend.endpage.net`。

```bash
cd /www/wwwroot/usr-iot-backend.endpage.net/backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp ../../deploy/backend.env.production.example .env
# 编辑 .env，填入真实配置
```

关键 `.env` 配置：

| 配置项 | 说明 |
|--------|------|
| `SECRET_KEY` | 随机长字符串，不能用默认值 |
| `DEFAULT_ADMIN_PASSWORD` | 管理员初始密码 |
| `CORS_ORIGINS` | `["https://usr-iot.endpage.net"]` |
| `MQTT_ENABLED` | `true` |
| `MQTT_BROKER_HOST` | `127.0.0.1`（EMQX 同机） |
| `MQTT_BROKER_PORT` | `1883` |

### 3.2 前端

宝塔面板 → HTML 纯静态站点，域名 `usr-iot.endpage.net`。

```bash
cd /www/wwwroot/usr-iot.endpage.net/web

# 创建生产环境变量
echo 'VITE_API_BASE=https://usr-iot-backend.endpage.net/api/v1' > .env.production

npm install
npm run build
```

宝塔站点根目录指向 `web/dist/`。

### 3.3 EMQX MQTT Broker

```bash
cd deploy
cp emqx.env.example emqx.env
# 编辑 emqx.env
docker compose -f emqx-docker-compose.yml up -d
```

防火墙开放 1883（MQTT 设备接入）。

### 3.4 微信小程序

本地编译后通过微信开发者工具上传发布。

小程序 `SERVICE_ORIGIN` 指向 `https://usr-iot-backend.endpage.net`。

## 4. 三端共用配置对照

| 端 | API 地址 | WebSocket |
|----|---------|-----------|
| Web 前端 | `https://usr-iot-backend.endpage.net/api/v1` | 自动推导 `wss://...` |
| 小程序 | `https://usr-iot-backend.endpage.net/api/v1` | `wss://usr-iot-backend.endpage.net/api/v1/ws/events` |
| 设备 MQTT | `usr-iot-backend.endpage.net:1883` | 不涉及 |

## 5. 更新部署

```bash
bash deploy.sh
```

一键完成：拉代码 → 装依赖 → 构建前端 → 重启后端。

## 6. 备份

数据库备份通过宝塔面板的计划任务完成，无需系统内置。
