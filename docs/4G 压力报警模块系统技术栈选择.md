# 4G 压力报警模块系统（4G Pressure Alarm Linkage System）技术栈选择

## 一、后端技术栈

| 层级           | 技术/工具               | 版本建议    | 用途               |
| :------------- | :---------------------- | :---------- | :----------------- |
| **框架**       | FastAPI                 | 0.110+      | Web API 框架       |
| **ASGI服务器** | Uvicorn                 | 0.27+       | 运行 FastAPI       |
| **数据库ORM**  | SQLAlchemy              | 2.0+        | 数据库操作         |
| **数据库驱动** | aiosqlite               | 0.19+       | SQLite 异步连接    |
| **认证**       | python-jose + passlib   | 3.3+ / 1.7+ | JWT + 密码哈希     |
| **设备通信**   | paho-mqtt               | 1.6+        | MQTT 客户端        |
| **定时任务**   | APScheduler             | 3.10+       | 离线检测、心跳检查 |
| **数据验证**   | Pydantic                | 2.6+        | 请求/响应模型      |
| **API文档**    | FastAPI 内置 Swagger UI | -           | 接口测试与文档     |
| **文件上传**   | python-multipart        | 0.0.9+      | 表单/文件上传      |
| **日志**       | Python logging          | 标准库      | 日志写入磁盘       |
